A content remixer using React

##

1. Paste in text which needs to be remixed
2. Click button to remix
3. Send request to an API end point
4. See result in output boxes
5. Add styling and other features as we go

## Features

1. React
2. TailWindCss
3. Vercel
4. Deepseek API
